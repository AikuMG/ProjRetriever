<div class="introduction">We are a group of UMBC undergraduates (and friends) that have come together to provide an interactive map of UMBC's Fine Arts Building. This project was designed by Angela Saccone who recruited Jennifer Saccone and Kwang Seo Yim for the User Experience element and Robert Schultz and Aiku Greenidge for the Software Development element. This project allows the user (you!) to click through each level of the Fine Arts Building to see a 2-D map and explore the area. As UMBC students, we know how easy it is to get lost in a building on campus, but this map will guide you to the right place.</div>
